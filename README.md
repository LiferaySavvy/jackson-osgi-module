# jackson-osgi-module
* These are dependency OSGi modules required for Liferay OSGI Rest Module
### Installation
* Clone current repo  
`git clone https://github.com/LiferaySavvy/jackson-osgi-module.git` 
* Copy the all jar files to portal server osgi/modules directoy
